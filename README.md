# Dart CLI 끝말잇기 게임
Made by: 하세린 (1학년 3반 30번 웹프로그래밍과)

## 프로젝트 목적
Dart를 사용하여 터미널에서 실행되는 간단한 끝말잇기 게임을 만들기 위해 제작했습니다.

## 주요 기능
- 컴퓨터가 제시한 단어의 마지막 글자로 시작하는 단어를 입력합니다.
- 입력한 단어가 규칙에 맞는지 확인합니다.
- 컴퓨터가 단어 목록에서 다음 단어를 찾아 출력합니다.
- 현재 컴퓨터는 정해진 단어 목록 안에서만 다음 단어를 출력합니다.

사용 가능한 컴퓨터 단어 목록:
과자, 자동차, 차고, 고양이, 이불, 불고기, 기차, 차표, 표지판, 자동문, 문어, 어머니, 이야기, 기린, 린스

## 실행 방법
터미널에서 프로젝트 폴더로 이동한 뒤 아래 명령어를 입력합니다.
```bash
dart main.dart
```
## 사용 예시
컴퓨터: 사과  
다음 단어 입력하세요: 과자  
입력한 단어는 과자 입니다.  
정답  
컴퓨터 단어: 자동차

## 사전 요구사항
- Dart가 설치되어 있어야 합니다.
- 터미널에서 명령어를 실행할 수 있어야 합니다.
